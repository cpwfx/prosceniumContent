Interfacepublic interface IWirable
Implementors AnimationController, AnimationTrack, AttributeXYZ, CompoundAttribute, MaterialStandard, Rotator, SceneNode, TransformElement, TransformElementLookAt, TransformElementMatrix, TransformElementRotate, TransformElementScale, TransformElementTranslate, TransformStack

Interface for classes that can have their attributes wired up. It creates a mechanism for lazy evaluation of values by tracking when values are current or need to be updated. The "smarts" for a Wirable class are defined in the evaluate and setDirty methods: The setDirty method is provided an attribute that is being dirtied. Dependant attributes need to be marked dirty. The evaluate method is provided an attribute that needs an updated value. It should calculate the value based upon its dependant attributes then set the value of the procided attribute and mark it clean.
