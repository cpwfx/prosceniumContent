Packagecom.adobe.binary
Classpublic final class GenericBinaryFormatDescription
InheritanceGenericBinaryFormatDescription Object

An object that describes a specific format implemented on top of Generic Binary. It specifies a namespace for the format and binds tags to actual class implementations. An instance of the class is passed when creating a new Generic Binary object or when reading from serialized binary data.
